A microservices app written in rust with an authentication and health-check service. A client is added to communicate with the authentication service. The app is containerized with docker, uses continuous integration with Github actions and has continuous deployment to a DigitalOcean droplet.

Objective of the project are:

1. Designing, building, and deploying microservices
2. Using gRPC to communicate between microservices
3. Monitoring the health of microservices
4. Setting up continuous integration & continuous deployment
5. Using session based authentication
6. Writing testable code
7. Organizing code using modules
8. Navigating and contributing to an existing code base
